# ❤️ Pulse

### Escutando os batimentos do seu sistema.

Um monitor de sistema desenvolvido em Python para acompanhar CPU, memória, disco, rede e processos em tempo real de forma simples, rápida e intuitiva.

## 📖 Sobre

O **Pulse** nasceu de uma pergunta simples:

> "O que diabos meu computador está fazendo agora?"

Enquanto o Gerenciador de Tarefas tenta resolver tudo com dezenas de números e abas, o Pulse busca apresentar apenas o que realmente importa.

Monitore o estado do sistema em tempo real, acompanhe o consumo de recursos e descubra rapidamente quando algum processo decide transformar sua máquina em um aquecedor portátil.

## ⚡ Instalação

Clone o repositório:

```bash
git clone https://github.com/ItzYago/Pulse.git
```

Acesse a pasta:

```bash
cd Pulse
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python pulse.py
```

## 🤔 Por que usar o Pulse?

Porque às vezes:

- O navegador abriu 37 abas.
- O Discord resolveu atualizar.
- O Windows está fazendo alguma coisa misteriosa.
- Um processo chamado `definitivamente_nao_e_um_virus.exe` apareceu do nada.

E você merece saber quem está consumindo os recursos da sua máquina.
